# Shale-Namma Pride

School Transparency & Pride Portal developed using HTML, CSS, JavaScript and Firebase.

## Features
- Daily Meal Updates
- Student Stars Section
- Facility Tour
- Parent Feedback Box
- Firebase Authentication
- Firestore Database
- Responsive Design

## Technologies Used
- HTML
- CSS
- JavaScript
- Firebase Authentication
- Firebase Firestore
- Vercel Deployment
